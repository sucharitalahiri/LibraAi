# LibraAI - Multi-Tenant SaaS Project Management Platform

## Features

- User Authentication (Signup/Login)
- JWT Authentication
- Role-Based Access Control
- Workspace Management
- Project Management
- Task Management
- Task Status Tracking
- Comments on Tasks
- Dashboard Statistics
- Search and Filter Functionality

## Tech Stack

### Frontend
- React.js
- React Router
- Axios
- CSS

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Bcrypt
